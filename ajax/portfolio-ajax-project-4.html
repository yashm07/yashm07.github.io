VaxNotifier - TOHacks 2021

Project Info:

VaxNotifier is a notification system that informs users of available vaccine appointments through an SMS text message. We created this app because during the start of the COVID-19 vaccination period, many people didn't know where to get their vaccines. Social media accounts were made to post the updates, but there were so many updates that no one had the time to check. This app was developed for TOHacks 2021.

Project Details:

  • Languages: Python, HTML/CSS
  • Tools: Twitter API, Courier API, Figma, Bootstrap
