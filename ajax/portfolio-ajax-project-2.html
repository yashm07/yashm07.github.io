E-Commerce React

Project Info:

DIYStore is an e-commerce application that I made for people to shop for their favourite hobby electronic items. Users can add their items to the cart, and check out their items. As of now, real payments are not processed. In the future, I'll work on using the Stripe API for payments. The goal of this project was to work on my React skills and get introduced to Firebase.

Project Details:

  • Languages: Javascript, HTML(JSX), CSS
  • Tools: React.js, React Bootstrap, Firebase
